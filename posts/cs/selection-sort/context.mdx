## 선택정렬

인자로 들어온 Array가 있을 때
Array를 순회하면서
가장 작은 숫자를 찾아 순회를 시작한 index와 swap하는 식으로 정렬을 한다

```js
function selection(arr) {
  for (let i = 0; i < arr.length; i++) {
    let lowest = i;
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[lowest] > arr[j]) {
        lowest = j;
      }
    }
    if (i !== lowest) {
      let temp = arr[i];
      arr[i] = arr[lowest];
      arr[lowest] = temp;
      console.log(arr);
    }
  }
  return arr;
}

selection([2, 6, 5, 3, 4, 1]); // [1,2,3,4,5,6]
```

여기서 최적화를 하려면 swap하는 부분을 함수화 시켜 빼는 방법이 있을 것 같다
